Portfolio

East High Support

East Highschool has a food pantry and donation programs it has a high population of homeless students or living in homes under the poverty line. The school didn't have a secure website to handle money donations and had some challenging UI.
To solve the problem we built a from scratch website to make it easy for people to donate food, clothing, money or time. This is a full stack React app with a admin dashboard to catalogue donation items, volunteer time and expenses in a MongoDB. This website is published on the east high website and I have made myself available to trouble shoot any problems that might arise in the future. This is an ongoing project and hopefully we will be able to bring it to other schools

Wizard Quest 4

A text based RPG clicker! This is a full stack web game ran with an Express server and MySql database and hosted on Heroku.
Explore a magical world and go on the Wizard's Quest which will have the player fight monsters, go on stealth missions, and save their love. In the game there are 3 main things to do: shop for assets, fight or use stealth to run away from monsters, and gather gold. If the game is too long the user can always resume by logging back in later. There are still some issues to be ressolved with the game so make sure to read the ReadME in the project's repo before playing.

Dutch Treat

A web app that provides the client with a movie based on their genre specification. It also gives a recepie pairing as well as which streaming platforms the movie is available.
It utilizes the OMDB and the FoodToFork API and MovieBox to call and get all the information. NOTE: the MovieBox API key was a trial beta run so it can't supply streaming platforms anymore.

Friend Finder

Friend Finder is mock up of a dating app. It is ran on an Express server. There are characters storied on the app for the client to "match" with based on answering a survey. As a bonus the client can also choose who they want to match with based on preference.

Tabletop RPG Character Tracker

This is a personal project I am working on. It is a MERN react app that will let players track table top game progress
